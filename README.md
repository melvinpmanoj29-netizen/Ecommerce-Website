# ME10XLUXE - Full Stack E-Commerce Platform

A modern full-stack e-commerce application built using:

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS
- Axios
- React Router

### Backend
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- AutoMapper
- FluentValidation
- JWT Authentication

### Features

- User Authentication
- Product Management
- Category Management
- Product Images (Cloudinary)
- Shopping Cart
- Quantity Update
- Remove Cart Items
- Order Placement
- Order Tracking
- Admin Order Management
- Responsive UI

## Project Structure

Ecommerce.API → ASP.NET Core Backend

ecommerce-client → React Frontend
